🏋️ Fitness Tracker Dashboard
A simple, interactive fitness tracker dashboard built using Next.js and Tailwind CSS. Users can log their daily workouts and weight trends, and view a summary of their fitness progress in real-time.
🚀 Features
•	📊 Track Workouts: Log workout type and duration with time stamps.
•	⚖️ Track Weight: Enter and visualize your weight over time using charts.
•	💾 Local Storage: All data is saved locally in the browser.
•	📈 Charts: View your weight trend in a responsive line chart.
•	🔥 Responsive Design: Works smoothly on desktop and mobile devices.
🛠️ Tech Stack
•	Next.js – React framework with SSR and routing
•	Tailwind CSS – For fast, responsive UI styling
•	Recharts – For visualizing weight trends
•	JavaScript (ES6) – Core programming logic
•	LocalStorage API – To store data locally

📦 Getting Started
1. Clone the repo:
   git clone https://github.com/muskanjain318/fitness-tracker.git
   cd fitness-tracker
2. Install dependencies:
   npm install
3. Run the dev server:
   npm run dev
4. Open your browser at http://localhost:3000
📌 Important Notes
Make sure your browser allows localStorage for data persistence.
No external database or backend is used.
